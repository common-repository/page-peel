== Description ==

This plugin adds a HTML5 pure CSS Page Peel effect to your WP powered web site.

Try it instantly on [tastewp.com](https://tastewp.com/new?pre-installed-plugin-slug=page-peel&redirect=__front&ni=true)!


== Installation ==

1. Upload contents of `page-peel.zip` to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create 2 files with your custom graphics: small.jpg (100x100 pixels) and large.jpg (400x400 pixels)
4. Configure plugin through Settings / Page Peel: add the links of your custom large.jpg and small.jpg and the target URL


== Frequently Asked Questions ==
